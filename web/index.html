How it works — Conway's Game of Life
Each cell has 8 neighbors (Moore neighborhood). Rules: (1) Any live cell with 2 or 3 live neighbors survives; (2) Any dead cell with exactly 3 live neighbors becomes alive; (3) All other cells die or remain dead. Toggle "Wrap edges" to treat borders as connected (toroidal) so patterns passing an edge reappear on the opposite side. This demo updates the grid every N ms (Speed). Size controls the number of cells per side.
